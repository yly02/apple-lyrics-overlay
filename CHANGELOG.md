# Changelog

## 1.0.1 - 2026-04-26

- Refined repository metadata for public distribution
- Added manual and automated release support for downloadable builds
- Improved release packaging and version tracking

## 1.0.0 - 2026-04-25

- Initial public-ready release
- Floating Apple Music desktop lyrics overlay for macOS
- Simplified Chinese translation for non-Chinese lyrics
- Menu bar controls for font, size, color, appearance, animation, and lock state
- Multiple lyric fallbacks including LRCLIB, Apple Music local lyrics, and lyrics.ovh
- Local credential storage and translation cache support
- Release packaging script and GitHub Actions release workflow
